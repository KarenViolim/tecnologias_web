Leitor de Áudio com jQuery

O que é o plugin?

  • É um plugin desenvolvido para realizar a reprodução de arquivos no formato MP3

Como faz para utilizá-lo?

  • É necessário importar a biblioteca jQuery e a biblioteca do CSS.
      <script src="js/jquery-3.3.1.js"></script>
      <script src="js/popper.min.js"></script>
      <script src="js/bootstrap.min.js"></script>
      <script src="js/audioplayer.js"></script>

  • Importe os arquivos do plugin jQuery AudioPlayer.js para a página. 
      <link href="css/bootstrap.min.css" rel="stylesheet">
      <link href="css/customizado.css" rel="stylesheet">
      <link rel="stylesheet" href="css/audioplayer.css">

  • Em seguida deve se incorporar um arquivo áudio HTML na página WEB.
  • O formato de áudio deve ser o .mp3 que é compatível com os navegadores.
      <div id="wrapper">
      <audio preload="auto" controls>
      <source src="audio/palmeiras.mp3">
      </audio>

  • Chame o plugin na tag de áudio.
      <script>
      $(function() {
      $('audio').audioPlayer();
      });
      </script>
      </div>
